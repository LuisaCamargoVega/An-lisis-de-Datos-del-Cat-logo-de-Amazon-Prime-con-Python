# An-lisis-de-Datos-del-Cat-logo-de-Amazon-Prime-con-Python
Análisis exploratorio de datos del catálogo de Amazon Prime usando Python, con limpieza, visualización y análisis de patrones en películas y series.
Herramientas utilizadas
Python
Pandas
NumPy
Matplotlib
Jupyter Notebook
📊 Principales hallazgos
Las series presentaron una presencia ligeramente superior frente a las películas.
Drama fue el género con mayor presencia en el catálogo, seguido por Thriller, Comedy y Action.
No se observó una relación fuerte entre la cantidad de votos y las puntuaciones IMDb.
Los países con mejores promedios IMDb mostraron resultados similares entre sí.
